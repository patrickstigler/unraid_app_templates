# Unraid App Templates

A curated collection of **Unraid Community Applications (CA) templates** for easy deployment of Docker containers.

[![Ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/W7W51C9X4H)

---

## Overview

This repository provides ready-to-use **Unraid XML templates** for deploying applications quickly and consistently.

Each template is designed to:

- simplify container setup  
- provide sensible defaults  
- reduce manual configuration effort  
- integrate cleanly into the Unraid ecosystem  

---

Each template includes:

- Docker image configuration  
- Port mappings  
- Volume bindings  
- Environment variables  
- Default settings
